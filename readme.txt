wpClickbank Help you to retrieve ClickBank Products with “specific keyword“, so any ClickBank Products that you love to promote.

== Description ==

wpClickbank just use easy fuction to get the digital products from clickbank, let check below for wpClickbank's utility. 

    * ClickBank Account : “Insert you CB Account ( automatic making your link )”
    * ClickBank Trace ID : “Optional Trace ID”
    * Keyword (comma separated) : “Specific Keyword ( automatic select ClickBank products ).”
    * Number Of Ads : ” How many ads do you want to display per page ?”
    * Width : ” Size of your ads “
    * Ads Position 
    * Ads Header

Testing and Working on WordPress 2.9.2,
